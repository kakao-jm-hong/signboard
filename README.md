# 결혼 청첩장 웹사이트

김승진 & 박지성의 결혼 청첩장 페이지입니다.

## 댓글 기능 설정 (Utterances)

이 페이지는 GitHub Issues를 댓글 저장소로 사용하는 Utterances를 사용합니다.

### 설정 방법

1. https://github.com/apps/utterances 방문
2. **Configure** 버튼 클릭
3. 저장소 선택: `kakao-jm-hong/signboard`
4. **Install** 클릭
5. 권한 부여 완료

설치 후 방문자들이 댓글을 남기면 자동으로 GitHub Issues에 저장됩니다.

## 로컬에서 실행

브라우저에서 `index.html` 파일을 열면 됩니다.

## GitHub Pages 배포

1. GitHub 저장소 Settings → Pages
2. Source를 "Deploy from a branch"로 설정
3. Branch를 "main"으로 선택
4. Save

배포 후 주소: `https://kakao-jm-hong.github.io/signboard/`
